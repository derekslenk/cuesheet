# Technology Stack

## Frontend
- **Next.js 15.4.1** - React framework with App Router
- **React 19.1.0** - UI library
- **TypeScript 5.8.3** - Type-safe JavaScript
- **Tailwind CSS 4.0.0** - Utility-first CSS framework
- **Custom CSS** - Glass morphism design with Solarized Dark theme

## Backend
- **Next.js API Routes** - Serverless API endpoints
- **Node.js** - Runtime environment

## Database
- **SQLite** - Embedded SQL database
- **sqlite3 5.1.7** - Node.js SQLite driver
- **Dynamic table naming** - Seasonal configuration (e.g., `streams_2025_summer_sat`)

## OBS Integration
- **obs-websocket-js 5.0.6** - WebSocket client for OBS Studio control
- **ws 8.18.0** - WebSocket library
- **bufferutil 4.0.8** - WebSocket performance optimization

## Development Tools
- **ESLint 9.20.0** - Code linting
- **Prettier** - Code formatting (via eslint-config-prettier)
- **Jest 30.0.4** - Testing framework
- **@testing-library/react** - React component testing
- **tsx 4.20.3** - TypeScript execution for scripts

## Build & Deployment
- **TypeScript Compiler** - Type checking
- **Next.js Build System** - Production optimization
- **Forgejo CI/CD** - Self-hosted continuous integration

## Design System
- **Solarized Dark** - Accessible colorblind-friendly theme
- **CSS Custom Properties** - Maintainable theming system
- **Glass Morphism** - Modern UI aesthetic with backdrop blur
- **WCAG AAA Compliance** - High contrast ratios (7.5:1+)
